Sets the financial assessment details based on the client's answers to analyze whether they possess the experience and knowledge to understand the risks involved with binary options trading.

For parameters information refer to SetFinancialAssessmentRequest. Throws a FinancialAssessmentException if API response contains an error

Implementation

static Future<SetFinancialAssessment> setAssessment(
  SetFinancialAssessmentRequest request,
) async {
  final SetFinancialAssessmentResponse response =
      await _api.call(request: request);

  checkException(
    response: response,
    exceptionCreator: (String message) =>
        FinancialAssessmentException(message: message),
  );

  return SetFinancialAssessment.fromJson(response.setFinancialAssessment);
}

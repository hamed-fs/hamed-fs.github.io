Future<SelfExclusion> fetchSelfExclusion (
  1. [GetSelfExclusionRequest request]

Allows users to exclude themselves from the website for certain periods of time, or to set limits on their trading activities.

This facility is a regulatory requirement for certain Landing Companies. For parameters information refer to GetSelfExclusionRequest. Throws a SelfExclusionException if API response contains an error

Implementation

static Future<SelfExclusion> fetchSelfExclusion([
  GetSelfExclusionRequest request,
]) async {
  final GetSelfExclusionResponse response = await _api.call(
    request: request ?? const GetSelfExclusionRequest(),
  );

  checkException(
    response: response,
    exceptionCreator: (String message) =>
        SelfExclusionException(message: message),
  );

  return SelfExclusion.fromJson(response.getSelfExclusion);
}
